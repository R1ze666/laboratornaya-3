3 Лабораторная 
В данной лабораторной осуществлялась работа с пакетами, для такой работы в языке go нужно создать специальный файл go.mod
Он позволяет управлять зависимостями в проекте и гарантирует совместимость библиотек.
После были созданы 3 файла Mathutils-с функцией для вычисления факториала, Strinutils-c функцией для переворота строки.
А также main в котором эти и другие функции были реализованны.
Сначала в main были импортированны функции из mathutils и stringutils
были использованы созданные пакеты для вычисления факториала и переворота строки.
Потом были написаны методы для создания массива и заполнения его 5-ю целыми числами, создания его среза и добавления и удадения элементов.
И метод, который создает срез и строк и наъодит самую длинную строку.
